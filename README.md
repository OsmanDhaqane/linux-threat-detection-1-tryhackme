Linux Threat Detection 1 - TryHackMe

This repository contains my walkthrough for the Linux Threat Detection 1 room on TryHackMe. It focuses on detecting initial access activity on Linux through SSH authentication logs, Nginx access logs, and auditd process tracing.

The room covers SSH key and password-based logins, brute-force detection, exposed web service abuse, command injection, reverse shell activity, and using process tree analysis to trace suspicious commands back to the parent application.
